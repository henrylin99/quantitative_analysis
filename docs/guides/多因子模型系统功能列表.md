# 多因子模型系统功能列表

## 项目概述

当前以真实可用能力为准，不再按完整量化平台对外描述。

本仓库目前适合作为学习、研究和二次开发的原型项目使用。以下列表仅统计当前主路径中已经保留或已明确下线 mock/demo 后的真实能力。

## 已实现

### 因子管理

- 因子列表、筛选与基础展示
- 内置因子基础链路
- 自定义因子表达式白名单校验
- 因子计算接口

### 模型管理

- 模型定义创建
- 真实训练任务轮询
- 模型列表与基础状态展示
- 模型删除前后端入口

### 股票评分

- 基础因子评分
- 基础 ML 评分链路
- 评分结果展示

### 组合优化

- 等权重
- 均值方差
- 风险平价
- 因子中性
- 优化结果导出
- 优化结果保存为真实投资组合持仓

### 投资组合

- 组合列表
- 组合详情
- 创建组合
- 删除组合
- 新增持仓
- 编辑持仓
- 删除持仓
- 组合页当前未开放再平衡入口

### 回测验证

- 基础回测配置
- 基础结果展示
- 前后端关键字段已做一轮对齐
- 失败时不再自动展示 mock 结果

## 部分实现

### 分析与报告

- 分析页已移除演示数据
- 报告管理与导出仍未形成稳定闭环

### 实时分析

- 主路径已清理 demo/mock 默认值
- 真实新闻源和部分推送能力尚未接入

### 文档与状态标识

- README 和核心整改文档已更新
- 仍有部分历史分析材料未同步清理

## 未实现或未开放

- Black-Litterman
- 行业约束优化
- 投资组合页面再平衡入口
- 完整报告管理后台
- 统一数据来源审计能力
- 统一功能状态标记机制

## 当前推荐使用路径

1. `python run_system.py`
2. 初始化数据库
3. 启动 Web 服务
4. 优先验证以下页面：
   - 因子管理
   - 模型管理
   - 股票评分
   - 组合优化
   - 投资组合
   - 回测验证

- 若某项能力没有出现在“已实现”部分，就不应默认视为可用。
- 历史文档若与本清单冲突，应以 `README.md`、整改清单和当前测试约束为准。
